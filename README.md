# 🪿 Гусь-Кликер (Windows 98 Style)

> 🔗 **Играть прямо сейчас:** [**Goose Clicker — нажми на гуся!**](https://saintmedium.github.io/GooseClicker/)

Кликер-игра, где вы кликаете по гусю, зарабатываете кристаллы и прокачиваете своего пернатого друга.

---

## 🎮 Геймплей

- **Кликайте по гусю** — зарабатывайте кристаллы 💎 и опыт ⭐
- **Улучшайте параметры** — сила клика, автоклик, шанс крита и многое другое
- **Прокачивайте гуся** — 9 уровней эволюции с уникальными картинками
- **Активируйте бусты** — временные усиления для максимальной эффективности

---

## 🖥️ Окна и управление

| Окно | Описание |
|------|----------|
| 🖥️ Мой компьютер | Покупка улучшений и активация бустов |
| 🎮 Игра | Основное окно с гусём и статистикой |
| 🗑️ Корзина | Сброс прогресса |

- **Перетаскивайте окна** за заголовок
- **Esc** — закрыть все окна
- **Пуск** — открыть меню (доступна Справка)
